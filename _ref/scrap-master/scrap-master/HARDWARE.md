# Hardware to be used for validation:

## Beaglebone Black (Momentus)
    - ARM Cortex-A8, 1GHz
    - 512MB RAM
    - 4GB Storage

## Jetson Nano (D-Phi - Momentus Vigoride 6 payload)
    - ARM Cortex-A57
    - Maxwell GPU (128 CUDA cores)
    - 4GB LPDDR4 (25 GB/s)
    - M.2 Key E

## Jetson Orin Nano
    - ARM Cortex-A78AE
    - Ampere GPU (1024 CUDA cores, 32 tensor cores)
    - 8GB RAM (LPDDR5, 102GB/s)
    - 8GB Storage
    - M.2 Key M, M.2 Key E
